# i18n + Settings System Implementation Plan

> **For agentic workers:** This plan is executed in two parts: **Phase 1 (foundation)** is done and smoke-verified by Claude during setup; **Phases 2–4** run as a parallel multi-agent Workflow overnight. Steps use checkbox (`- [ ]`) syntax. This project has **no unit-test framework** (see `docs/conventions/gameplay-code.md`); the verification gate is the **headless smoke test** + a **key-audit script**, not pytest-style tests.

**Goal:** Ship a Settings menu (Language / Fullscreen / Master Volume) with a complete English↔简体中文 translation of all UI and content, switchable at runtime and persisted.

**Architecture:** Godot built-in `TranslationServer` + `tr()` + CSV translation files. Content JSON stays English (source of truth); display routes through `Settings.t(key, english_fallback)` so a missing zh value degrades to English. A global CJK font (`assets/fonts/NotoSansCJKsc-Regular.otf`) makes Chinese render. A new `Settings` autoload owns `user://settings.json` and applies locale on boot.

**Tech Stack:** Godot 4.6, GDScript, CSV translations, `scripts/smoke_test.sh` (headless `DataValidator`). Smoke binary: `GODOT_BIN="C:/Program Files/Godot/Godot.exe"`.

**Spec:** `docs/superpowers/specs/2026-05-29-i18n-settings-design.md` (includes the translation glossary — all translators MUST follow it).

### Key scheme (deterministic — both translators and wirers follow it)
- UI: `UI_<AREA>_<NAME>` (e.g. `UI_BATTLE_END_ROUND`, `UI_LOOT_PROCEED`, `SETTINGS_LANGUAGE`). Formatted: `String` with `{n}` placeholders, e.g. `UI_MAP_SCAVENGED_GOLD = "Scavenged {n} gold."`, used `tr("UI_MAP_SCAVENGED_GOLD").format({"n": gold})`.
- Content (from existing id): `CARD_<id>_TITLE`/`_DESC`, `ENEMY_<id>_NAME`, `RELIC_<id>_TITLE`/`_DESC`, `EQUIP_<id>_NAME`/`_DESC`, `SET_<id>_NAME`, `UPGRADE_<id>_NAME`/`_DESC`, `HERO_<id>_NAME`/`_DESC`.
- Locale codes: `en` and `zh`. CSV header row is exactly `keys,en,zh`.

```gdscript
## App settings (language / display / audio) persisted to user://settings.json.
## Autoloaded BEFORE the main scene so the locale is applied on the first frame.
## Also hosts t(): the content-translation helper with English fallback.
extends Node

const SAVE_PATH := "user://settings.json"
const DEFAULT_LANGUAGE := "en"

var language: String = DEFAULT_LANGUAGE
var fullscreen: bool = false
var master_volume: float = 1.0


func _ready() -> void:
	load_settings()
	_apply_all()


func load_settings() -> void:
	if not FileAccess.file_exists(SAVE_PATH):
		return
	var f := FileAccess.open(SAVE_PATH, FileAccess.READ)
	if f == null:
		return
	var data = JSON.parse_string(f.get_as_text())
	if typeof(data) != TYPE_DICTIONARY:
		return
	language = str(data.get("language", DEFAULT_LANGUAGE))
	fullscreen = bool(data.get("fullscreen", false))
	master_volume = float(data.get("master_volume", 1.0))


func save_settings() -> void:
	var f := FileAccess.open(SAVE_PATH, FileAccess.WRITE)
	if f == null:
		push_warning("Settings: could not write %s" % SAVE_PATH)
		return
	f.store_string(JSON.stringify({
		"language": language, "fullscreen": fullscreen, "master_volume": master_volume,
	}))


func _apply_all() -> void:
	TranslationServer.set_locale(language)
	_apply_fullscreen()
	_apply_volume()


func set_language(loc: String) -> void:
	language = loc
	TranslationServer.set_locale(loc)
	save_settings()


func set_fullscreen(on: bool) -> void:
	fullscreen = on
	_apply_fullscreen()
	save_settings()


func set_master_volume(v: float) -> void:
	master_volume = clampf(v, 0.0, 1.0)
	_apply_volume()
	save_settings()


func _apply_fullscreen() -> void:
	DisplayServer.window_set_mode(
		DisplayServer.WINDOW_MODE_FULLSCREEN if fullscreen else DisplayServer.WINDOW_MODE_WINDOWED
	)


func _apply_volume() -> void:
	var bus := AudioServer.get_bus_index("Master")
	if bus >= 0:
		AudioServer.set_bus_volume_db(bus, linear_to_db(maxf(master_volume, 0.0001)))


## Content-translation helper. Returns the localized string for `key`, or
## `fallback` (the English source) when the key has no translation — so a
## missing zh value degrades to English, never a raw key.
func t(key: String, fallback: String = "") -> String:
	var s := tr(key)
	if s == key:
		return fallback if fallback != "" else key
	return s
```
