# American Comic Style Trial

Exploratory preview only. These assets are not wired into the game.

Goal: test whether Cowboy Bill and the wasteland UI world can move from the current Robo-Cowboy concept-sheet look toward a bolder American comic-book art direction.

Trial prompt constraints:

- Bold American sci-fi western comic art.
- Thick confident black ink outlines.
- Large clean shapes and dramatic cel shading.
- Saturated but controlled desert palette.
- No pixel art, no halftone dots, no tiny texture noise, no painterly realism.
- Preserve Cowboy Bill identity: one orange camera eye, cowboy hat, red scarf, patched coat, skull belt, robot limbs.

Files:

- `bill-character-comic-trial.png`
- `battle-background-comic-trial.png`
- `charged-shot-card-comic-trial.png`

Initial read:

- Character/card impact is stronger than the current softer concept-sheet style.
- The model still tends to add too much small wasteland detail unless prompts explicitly say broad shapes, low texture, and minimal line clutter.
- If adopted, the production rule should be renamed and tightened before replacing in-game assets.
