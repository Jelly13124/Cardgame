# Offbeat Card Rework Trial

Three card-art previews installed into the game to replace missing card images. These are flat vector-style previews aligned to the new Offbeat Adult Sci-Fi Cartoon Wasteland direction.

The `*_card_mockup.png` files show those illustrations inside the current in-game card UI. The `*_offbeat_card_concept.png` files and `offbeat_card_rework_concept_sheet.png` are a review-only concept for a non-pixel card frame; they do not replace the global card UI yet.

`offbeat_card_ui_square_cost_no_rarity_label_sheet.png` applies the requested UI direction: square cost badge, no rarity text labels, and rarity communicated only by the art-border color.

`offbeat_card_ui_larger_art_clean_skill_lane_sheet.png` refines that direction with a larger art window and a clean centered `SKILL` badge lane between the description box and the bottom frame.

`offbeat_card_ui_filled_title_and_type_lanes_sheet.png` removes the remaining dead space: the title strip fills the full top lane next to the square cost, and the `SKILL` strip fills the whole area from the description box to the bottom frame.

`offbeat_card_ui_skill_docked_to_text_and_layer_boundary_sheet.png` docks the centered `SKILL` badge to the structure: its top edge touches the description box and its bottom edge touches the light-brown/dark-brown layer boundary.
