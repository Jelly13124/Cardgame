暴击 & 幸运 重做

把暴击从「只有暴击弹夹才能触发」改成 幸运驱动的基础机制，这样双发流也能用暴击卡。

情形暴击率
任何角色（基础）幸运 × 2%
装了暴击弹夹5% 底 + 幸运 × 4%（幸运翻倍）
暴击倍率默认 1.5×（梭哈为 2×）
暴击上限无上限（移除现有 40% cap）—— 高幸运可冲到 100%+ 必暴

实装影响：会推翻之前「暴击=Bill专属 / 从装备移除暴击率」的改动 —— 暴击要改回走基础攻击路径（不再只靠 crit_clip 遗物触发）。装备暴击率词缀：恢复 ✅（重新加回 affix 池）。

提案卡

备弹 / Load Up

技能 Skill1 费装弹体系
将 2 张「装弹」置入你的手牌。
让双发流能在弹夹给的 2 张装弹之外，临时多造一批弹药打爆发回合。

走运 / Hot Streak

能力 Power1 费暴击体系
本场战斗，你每次暴击就获得 2 金币。
需机制：暴击时触发钩子。（改成给金币而非幸运，避开暴击→幸运→更高暴击的滚雪球。）

致命失血 / Hemorrhage

能力 Power2 费Rare流血 × 暴击
你的流血伤害也能暴击。
需机制：流血结算时按暴击率掷骰、命中则 ×倍率。把流血流和暴击流缝在一起。

换弹掩护 / Covering Reload

能力 Power1 费装弹体系
本场战斗，每当你打出「装弹」时，获得 3 点格挡。
需机制：装弹触发钩子（restore_attack_allowance 时给 block）。给装弹流一条防御向收益。

解剖 / Dissect

攻击 Attack1 费流血 × 智力
施加等同于双倍智力的流血（2 × INT）。
需机制：扩展 apply_bleed_scaled 支持属性倍率（amount:0, attr:intelligence, attr_mult:2）。纯智力缩放放血。

凝血 / Coagulate

技能 Skill1 费流血 → 格挡
获得等同于所有敌人当前流血层数总和的格挡（1:1）。
需机制：统计全场敌人流血层数 → 转格挡。比例 1:1（格挡只存一回合，高也无妨）。

梭哈 / All In

能力 Power1 费暴击体系
本场战斗：你的暴击造成 2 倍伤害，但你未暴击的攻击造成 0 伤害。
需机制：power 改写暴击倍率为 2×，且非暴击攻击伤害归零。全有或全无 —— 逼你把暴击堆满（配无上限暴击 + 暴击弹夹）。

已定 / 已实装的相关基础（供参考）

  • 装弹 / Reload（已实装）：0费，抽1 + 刷新攻击次数（到上限不叠加）+ 消耗。双发弹夹拿到时塞 2 张。
  • 双发弹夹（已实装）：攻击重放1 + 每回合限1攻击。
  • 流血卡（已实装）：切割、切割伤口、回旋锯刃；遗物：锯齿倒刺、磨尖废铁、残暴伺服。
